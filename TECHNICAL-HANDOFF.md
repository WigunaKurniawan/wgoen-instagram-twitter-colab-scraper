# Technical Handoff

## Stack

Untuk Instagram saya pakai Python + Instaloader. Untuk Twitter/X saya siapkan jalur yang lebih stabil lewat input manual atau browser-assisted, jadi tidak bergantung penuh pada public search yang sering berubah.

## Deliverable

- Notebook Google Colab utama.
- Script Instagram otomatis berbasis Instaloader.
- Script Twitter/X untuk normalisasi input manual/browser-assisted.
- Mode `snscrape_experimental` sebagai eksperimen terpisah.
- Sample CSV dan JSON yang aman dibagikan.

## QC Checklist

- Tidak ada cookie, password, token, atau session file.
- Parameter notebook tetap generik.
- Twitter/X tidak diposisikan sebagai full automation yang selalu stabil.
- Sample output bersifat sintetis/sanitized.
- Lisensi publik meminta izin tertulis WGOEN untuk penggunaan komersial.
